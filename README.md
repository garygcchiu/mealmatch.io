# MealMatch.io

Project for Website

## Getting Started

Install dependencies:

```bash
npm i
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## About

- Next.js application
- CI/CD configured with AWS CodePipeline and CodeBuild
